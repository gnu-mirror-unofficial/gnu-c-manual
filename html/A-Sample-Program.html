7 A Sample Program

To conclude our description of C, here is a complete program written in C, consisting of both a C source file and a header file. This program is an expanded version of the quintessential “hello world” program, and serves as an example of how to format and structure C code for use in programs for FSF Project GNU. (You can always download the most recent version of this program, including sample makefiles and other examples of how to produce GNU software, from ftp.gnu.org.)

This program uses features of the preprocessor; for a description of preprocessor macros, see The C Preprocessor, available as part of the GCC documentation.
