1.2 Keywords

Keywords are special identifiers, reserved for use by the progamming language itself. You cannot use them for any other purpose.

Here is a list of keywords recognized by ANSI C89:

  • auto
  • break
  • case
  • char
  • const
  • continue
  • default
  • do
  • double
  • else
  • enum
  • extern
  • float
  • for
  • goto
  • if
  • int
  • long
  • register
  • return
  • short
  • signed
  • sizeof
  • static
  • struct
  • switch
  • typedef
  • union
  • unsigned
  • void
  • volatile
  • while
