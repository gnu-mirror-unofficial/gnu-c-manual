The GNU C Reference Manual

[ English ]

The GNU C Reference Manual is a guide to the C programming language as implemented by the GNU C compiler. This manual is still a work in progress.

The scope of the manual is to document the GNU C language proper; that is, the various data types, expressions, statements, and so forth. You can learn more about the standard library functions in The GNU C Library Reference Manual. This is not a tutorial for beginners; you will probably find this manual more useful if you have some programming experience, preferably in C or a similar language.

You can view a recent draft of this manual in HTML format. Please keep in mind that this is not complete yet; there are still some sections that are missing, and there may be mistakes in what has been written so far.

You can also download the source code for the most current revision from Savannah.
